# Эмулятор Машины Тьюринга

## turingmachine.h

Описаны методы класса `TuringMachine`.

### Что хранит класс `TuringMachine`

- `m_tapeAlphabet` — алфавит входного слова
- `m_extraAlphabet` — дополнительные символы
- `m_fullAlphabet` — полный алфавит машины
- `m_tape` — текущая лента
- `m_initialTape` — исходное состояние ленты для сброса
- `m_program` — таблица команд
- `m_states` — список состояний
- `m_headPos` — текущая позиция головки
- `m_initialHeadPos` — начальная позиция головки
- `m_currentState` — текущее состояние
- `m_initialState` — начальное состояние
- `m_blank` — пустой символ
- `m_running` — флаг запущенной машины
- `m_lastMove` — последнее движение головки (`L`, `R`, `S`)

### Основные методы

- `setAlphabets(...)` — задаёт алфавиты
- `setInputWord(...)` — записывает слово на ленту
- `clearProgram()` — очищает программу
- `setStates(...)` — задаёт состояния
- `setCommand(...)` — добавляет команду
- `step()` — выполняет один шаг
- `reset()` — сбрасывает машину
- `symbolAt(...)` — возвращает символ в клетке
- `headPosition()` — возвращает позицию головки
- `currentState()` — возвращает текущее состояние
- `states()` — возвращает список состояний
- `fullAlphabet()` — возвращает полный алфавит
- `tapeAlphabet()` — возвращает алфавит ленты
- `blankSymbol()` — возвращает пустой символ
- `isRunning()` / `setRunning(...)` — работа с флагом запуска
- `hasAnyHaltCommand()` — проверяет наличие команды остановки
- `lastMove()` — возвращает последнее движение головки

## turingmachine.cpp

Код методов класса `TuringMachine`.

1. `setAlphabets`
	* Сохранение алфавита входного слова
	* Дополнительный алфавит
	* После изменения алфавита очищение ленты и перезапуск машины

2. `setInputWord`
	* Проверка на корректность входного слова
	* Удаление старого содержимого ленты
	* Запись слова на ленту
	* Сброс позиции головки и состояния
	* Сигнал изменения

3. `clearProgram`
	* Очищение всех комманд

4. `setStates`
	* Задаёт список состояний

5. `setCommand`
	* Создает ключ вида `(состояние, символ)` и закрепляет за ним нужную комманду

6. `step`
	* Делает 1 шаг машины
	* Считывает символ под головкой
	* Ищет подходящую команду
	* Записывает новый символ
	* Сдвигает головку
	* Меняет состояние
	* Сохраняет последнее движение
	* При необходимости останавливает машину

7. `reset`
	* Выставляет все значения в исходные

8. `symbolAt`
	* Возвращает символ по индексу клетки
	* Если клетка пуста, возвращает пустой символ

9. `headPosition`
	* Возвращает текущую позицию головки

10. `currentState`
	* Возвращает текущее состояние машины

11. `states`
	* Возвращает список состояний

12. `fullAlphabet`
	* Возвращает полный алфавит машины

13. `tapeAlphabet`
	* Возвращает только алфавит входного слова

14. `blankSymbol`
	* Возвращает пустой символ

15. `isRunning`
	* Проверяет, запущена ли машина

16. `setRunning`
	* Меняет значение флага запуска

17. `hasAnyHaltCommand`
	* Проверяет, есть ли хотя бы одна команда остановки

18. `makeKey`
	* Создаёт строковый ключ вида `q0|a`

19. `lastMove`
	* Возвращает последнее движение головки

## secondwindow.h

Описан класс `SecondWindow`, который отвечает за основное рабочее окно программы.

### Что хранит `SecondWindow`

- указатель на интерфейс `ui`
- объект машины `m_machine`
- таймер `m_timer` для автоматического выполнения
- анимацию каретки `m_caretAnimation`
- `m_stateCount` — количество состояний
- `m_stepIntervalMs` — скорость работы
- `m_viewOffset` — индекс левой видимой клетки
- `m_caretScreenIndex` — номер видимой ячейки под кареткой
- `m_pendingDirection` — ожидаемое направление движения
- `m_stepInProgress` — выполняется ли сейчас анимация шага

### Основные методы

- `setAlphabets(...)` — принимает алфавиты из главного окна
- `rebuildProgramTable()` — перестраивает таблицу программы
- `parseCommandText(...)` — разбирает текст команды из ячейки
- `loadProgramFromTable()` — загружает программу из таблицы в машину
- `updateTapeLabels()` — обновляет визуальную ленту
- `updateUiTextOnly()` — обновляет только текстовую часть интерфейса
- `updateUi()` — обновляет весь интерфейс
- `highlightCurrentStateRow()` — подсвечивает текущую строку таблицы
- `setEditingEnabled(...)` — блокирует или разблокирует элементы интерфейса
- `placeCaretOverCell(...)` — ставит каретку над нужной ячейкой
- `animateCaretToCell(...)` — анимирует движение каретки
- `finishStepAnimation()` — завершает шаг после анимации
- `generateStates(...)` — создаёт имена состояний `q0, q1, ...`
- `sortedAlphabetList()` — формирует порядок столбцов таблицы
- `isMoveCommand(...)` — проверяет, является ли строка командой движения

## secondwindow.cpp

Код методов класса `SecondWindow`.

### Основные части

1. Конструктор окна
	* Создание интерфейса
	* Настройка кнопок окна
	* Создание анимации каретки
	* Подключение таймера и сигналов машины
	* Начальная блокировка элементов интерфейса

2. `on_pushButton_2_clicked`
	* Возврат в главное окно изменения алфавитов
	* Скрытие рабочего окна без уничтожения данных

3. `setAlphabets`
	* Передаёт алфавиты машине
	* Перестраивает таблицу программы
	* Разблокирует элементы интерфейса

4. `on_pushButtonSetWord_clicked`
	* Считывает входную строку
	* Передаёт её машине
	* Настраивает видимую часть ленты
	* Ставит каретку на центральную ячейку

5. `on_pushButtonStep_clicked`
	* Выполняет один ручной шаг
	* Загружает таблицу команд
	* Вызывает `step()` у машины
	* При необходимости запускает анимацию каретки

6. `on_pushButtonStart_clicked`
	* Загружает программу
	* Проверяет наличие остановки
	* Запускает автоматический режим

7. `on_pushButtonStop_clicked`
	* Останавливает таймер
	* Возвращает возможность редактирования

8. `on_pushButtonReset_clicked`
	* Сбрасывает машину
	* Останавливает таймер и анимацию
	* Возвращает ленту и каретку в начальное положение

9. `onAutoStep`
	* Выполняет шаг по таймеру
	* Используется в автоматическом режиме

10. `on_pushButtonSpeedUp_clicked` / `on_pushButtonSpeedDown_clicked`
	* Изменяют скорость работы машины
	* При необходимости сразу меняют интервал таймера

11. `rebuildProgramTable`
	* Перестраивает таблицу состояний и символов
	* Сохраняет старые данные при расширении алфавита
	* Полностью очищает таблицу, если старые символы исчезли

12. `parseCommandText`
	* Поддерживает несколько форматов записи команд:
		- `R`
		- `q1`
		- `a`
		- `R,q1`
		- `a,R`
		- `a,q1`
		- `a,R,q1`
		- `!`
	* Заполняет структуру `Command`

13. `loadProgramFromTable`
	* Считывает все непустые ячейки таблицы
	* Проверяет корректность команд
	* Загружает их в машину

14. `updateTapeLabels`
	* Показывает 11 видимых клеток ленты
	* Подсвечивает клетку под кареткой

15. `highlightCurrentStateRow`
	* Подсвечивает строку состояния, в котором сейчас находится машина

16. `placeCaretOverCell`
	* Ставит треугольник-каретку над нужной видимой ячейкой

17. `animateCaretToCell`
	* Плавно перемещает каретку между соседними ячейками

18. `finishStepAnimation`
	* Завершает шаг после анимации
	* При достижении края сдвигает ленту на `2/3` видимой части

## Логика работы программы

1. В главном окне пользователь вводит:
	* алфавит входного слова
	* дополнительный алфавит

2. После этого открывается рабочее окно:
	* можно задать строку
	* можно заполнить таблицу команд
	* можно запускать машину пошагово или автоматически

3. Таблица программы строится по принципу:
	* строки — состояния
	* столбцы — символы алфавита

4. При нажатии **Шаг**:
	* выполняется одна команда
	* каретка перемещается
	* при достижении края лента сдвигается

5. При нажатии **Старт**:
	* машина начинает выполнять шаги автоматически по таймеру

## Поддерживаемые сокращённые форматы команд

- `R` — двигаться вправо
- `L` — двигаться влево
- `S` — не двигаться
- `q1` — перейти в состояние `q1`
- `a` — переписать текущий символ на `a`
- `!` — остановка
- `R,q1` — сдвиг и переход
- `a,R` — запись символа и сдвиг
- `a,q1` — запись символа и переход
- `a,R,q1` — полный формат команды

---

## Особенности реализации

- Лента хранится не как массив фиксированной длины, а как `QMap<int, QChar>`
- Пустые клетки не хранятся явно, а возвращаются как `blankSymbol`
- Видимая часть ленты состоит из 11 ячеек
- Каретка рисуется отдельным `QLabel`
- При расширении алфавита таблица старается сохранить уже введённые команды

## Пример программы

Для алфавита `abc` можно задать такую программу:

### Состояние `q0`
- `a -> R`
- `b -> q1`
- `c -> R`
- `Λ -> !`

### Состояние `q1`
- `b -> a,q0`

Такая программа проходит по слову и заменяет символ `b` на `a`.
