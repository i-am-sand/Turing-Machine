# Эмулятор Машины Тьюринга

## turingmachine.h

Описаны методы класса `TuringMachine`



## turingmachine.cpp

Код методов класса TuringMachine

1. setAlphabets
	* Сохранение алфавита входного слова
	* Дополнительный алфавит
	* После изменения алфавита очищение ленты и перезапуск машины

2. setInputWord
	* Проверка на корректность входного слова
	* Удаление старого содержимого ленты
	* Запись слова на ленту
	* Сброс позиции головки и состояния
	* Сигнал изменения

3. clearProgram
	* Очищение всех комманд

4. setStates
	* Задаёт список состояний

5. setCommand
	* Создает ключ вида (состояние, символ) и закрепляет за ним нужную комманду

6. step
	* Делает 1 шаг машины

7. reset
	* Выставляет все занчения в исходные
